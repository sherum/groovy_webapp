About

The Writer's Sauce is a beta implementation of the vfvt.org API. The current version (v0.3) is a minimum viable product to test the utility of a scaffolding tool for fiction authors. The current tool supports:

Simple email/password authentication

One-to-one author-user account

Free use for beta accounts
